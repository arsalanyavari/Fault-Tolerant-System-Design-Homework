Using the partition theorem, we will solve the problem. If a failure is covered
(i.e., fault detected and using the spare component), since our spare
component is a hot spare and works in synchronization with the primary, the
reliability formula will be the same as the reliability of a parallel system.
R(Tsys | I=1) = Rp(t) + Rs(t) - Rp(t)*Rs(t)
If no fault is detected, then the system's reliability is equal to the reliability of
the primary component.
R(Tsys | I=0) = Rp(t)
So the reliability of the entire system is equal to:
R(Tsys) = C * R(Tsys | I=1) + (1 - C) * R(Tsys | I=0)
